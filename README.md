# TextSkill

TextSkill содержит правила для промтов. Запрещённые фразы, синтаксические паттерны, живые альтернативы. Для русского языка. Claude и ChatGPT отдельно: маркеры разные.

---

## Структура

```
core/           # универсальные паттерны — для любой модели
├── writing-rules.md
└── audit.md

claude/         # дополнение для Claude
├── writing-rules.md
└── audit.md

chatgpt/        # дополнение для ChatGPT
├── writing-rules.md
└── audit.md
```

## Промты

Замени `[модель]` на `claude` или `chatgpt`.

**Пост для Telegram**
```
Используй правила из core/writing-rules.md и [модель]/writing-rules.md. Соблюдай раздел «Правила для Telegram-постов».

Тема: [тема]
Канал: [о чём канал, кто читает]
Тон: [разговорный / деловой / личный]
```

**Статья**
```
Используй правила из core/writing-rules.md и [модель]/writing-rules.md. Соблюдай раздел «Правила для статей».

Тема: [тема]
Тезис: [главная мысль]
Аудитория: [кто читает, что уже знает]
Объём: [примерно слов]
Начни с конкретной сцены или момента, не с тезиса.
```

**Переписать текст**
```
Используй правила из core/writing-rules.md и [модель]/writing-rules.md.
Перепиши текст: убери AI-паттерны, сохрани смысл и структуру.

Текст:
---
[текст]
---
```
